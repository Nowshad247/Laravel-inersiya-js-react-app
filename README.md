# Skills Development Center Portal

This project is designed to manage a Skills Development Center through an official Course Management Portal. It provides administrators and staff with tools for student and course management, lead tracking, role-based access control, and portal configuration.

## Core Features

- Student Management System with full CRUD functionality.
- Unique Student Super Key ID generation for certificate verification and authentication.
- Course and Batch Management, including Course Codes and Batch Codes.
- Lead Management System for tracking and organizing prospective students.
- Lead Follow Up and related management functionalities to improve conversion and communication.
- Role Management System with access control for different user types.
- Website Settings and Portal Configuration features for complete platform administration.

## Overview

The portal centralizes the administrative workflow for a skills development center by allowing staff to manage student records, courses, batches, and leads within a single platform. Secure role-based permissions ensure that users only access the features they need.
